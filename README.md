ReadSteed is a beautifully designed, modern Python desktop application that trains you to read significantly faster. By flashing text one word at a time at the Optimal Recognition Point (ORP), it absolutely eliminates the time your eyes waste moving across a page.

---

## ✨ Features

- 🏎️ **Dynamic Speed Tuning:** Instantly adjust your Reading Speed (WPM) via a dynamic slider.
- 🎯 **Advanced ORP Highlighting:** Automatically centers your eye exactly where it needs to be with contrasting colors.
- 🌙 **Focus & Themes:** Clean light/dark modes paired with a distraction-free "Focus Mode."
- 🗂️ **Multi-Reading Memory:** Logs, tracks, and sorts an infinite number of reading sessions neatly into a collapsible sidebar.
- 🚦 **Real-time Metrics:** Constant live progress bar tracking to keep you motivated.

---

## 🚀 Easy Setup Guide

Follow these quick steps to get RSVP Reader running locally on your machine.

### 1. Clone the repository
```bash
git clone https://github.com/yourusername/readsteed.git
cd readsteed
```

### 2. Install the requirements
Ensure you have Python installed. Then, install the required UI library.
```bash
pip install -r requirements.txt
```

### 3. Run the App
Launch the main application using Python:
```bash
python main.py
```

*(Windows users: You can also just double-click the `Run_RSVP.bat` script!)*
